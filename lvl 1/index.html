Level 1: Style the Box!

In this level, your goal is to style the box inside the canvas area using only CSS. You can change the box's style, color, and position. Try it out!

Example Styles

Preview Style Description
background-color: tomato; Change the color to a fun red-orange.
border-radius: 10px; Round the corners a bit for a softer look.
background: linear-gradient(to right, red, orange); Use a gradient for a colorful effect.
border: 2px dashed blue; Add a dashed border to the box.
box-shadow: 2px 2px 5px ... Make the box pop out with a soft shadow.
