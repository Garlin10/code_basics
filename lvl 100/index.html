Level 5: Grow the Tail 🐍

You've built a moving box — now it's time to grow a tail! In this level, you won't modify the movement logic. Instead, your job is to track the box’s movement and draw a green tail behind it.

Open tail.js. You’ll find a function called updateTail(top, left) — this is called automatically every time the box moves.

🧠 Your mission:

  • Save the box’s previous position into an array
  • Only keep the last 5 positions (like a snake body)
  • Create new green <div class="tail"> segments at each saved position
  • Make sure you don’t overwrite the red head (.box)

💡 Hints:

  • Use positions.push({ top, left }) to store data
  • Use positions.shift() to remove the oldest entry
  • You can remove elements using element.remove()
  • Use document.createElement('div') to create new tail pieces

// Get previous box position:
const box = document.querySelector('.box');
const prevTop = parseInt(box.style.top);
const prevLeft = parseInt(box.style.left);
