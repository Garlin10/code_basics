Level 3: Move the Box!

Numbers have appeared on the board! In this level, you'll write JavaScript to move your red box around the grid. The canvas is divided into 20Ã—20 pixel squares. Try moving the box step by step using pixel pushes.

Hereâ€™s a helper function that moves the box by a small step, with a short delay so you can see each move in action: 


  // Move the box step-by-step with a delay
  async function moveBoxBy(x, y, delay = 500) {
    const box = document.querySelector('.box');
    const top = parseInt(box.style.top || "0");
    const left = parseInt(box.style.left || "0");
  
    // Wait before moving
    await new Promise(resolve => setTimeout(resolve, delay));
  
    box.style.top = (top + y) + "px";
    box.style.left = (left + x) + "px";
  }

To use it, you must create an async function and use await to wait between steps. Like this: 


  async function moveToTarget() {
    await moveBoxBy(20, 0); // right
    await moveBoxBy(0, 20); // down
    await moveBoxBy(0, 20); // down
  }
  
  moveToTarget();
