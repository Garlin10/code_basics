Level 7: Knight Rider Lights 💡

Your red box is now a scanner light — like KITT from Knight Rider! Make it sweep left to right... then back again. 🔁

🔢 Your Tasks:

  1. The red light already moves automatically — nothing to do there ✅
  2. Each time it moves, it leaves a red dot (that part is also done ✅)
  3. Your mission: Keep only the last 3 dots on the screen!

🧠 What You Need to Do

Open trail.js — there is a function called addDot(dot). You’ll complete it!

Inside addDot(dot), do the following:

  • Use push() to add each new dot to the end of the trail array
  • When the array gets longer than 3, remove the oldest dot:
    • Use trail[trail.length - 4].remove() to remove the oldest dot from the screen
    • Then call trail.shift() to remove it from the array

💡 Concepts You’ll Practice

  • push() – add a new item to the array
  • length – check how many items are in the array
  • array[index] – get the oldest dot
  • .remove() – remove that dot from the screen
  • shift() – clean up the array

✨ Bonus Challenge

  • Try showing 5 dots instead of 3
  • Make it go faster or slower
  • Can you make it bounce vertically too?
